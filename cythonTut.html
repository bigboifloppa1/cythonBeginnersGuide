A beginner's guide to Cython


In order to use Cython, you must first install Cython (obviously).
In your command prompt, type in

pip install cython


Ok now, we can start coding straight away after the installation is done! Just follow my steps.

#main.pyx
import cython

x = 20 + 20
print(x)


Highly recommend you to copy it manually as it will give you more experience as a programmer!


By the way, note that I did not use the Cython functions yet? That is because I am going to show you some comparison between pure Python vs Cython

Now, we will see! :)


So now lets run this python file and..

40
0.0388033069903031s


Oh yeah I forgot I also imported time sorry about that. But that doesn't really matter now does it?


Ok now, we need to create setup.py file. Don't worry if you're not familiar with it, probably just try to copy & paste it or type it manually

from setuptools import setup
from Cython.Build import cythonize

setup(
  ext_modules = cythonize("main.pyx")
)


Ok, now this command line should be simple

python setup.py build_ext --inplace


It should say something like Compiling main.pyx or something.

Ok now we're ready to run the code! We have 2 options,

1. Directly from the Python's shell
2. In another python file


Well! Why not try 'em both?


#Directly from the Python's shell
$ python
>>> import main
40
0.00015507699572481215


That's a whopping 200-300x faster!


Ok, now we will try it in another Python file

#test.py
import main
print(main)


Well, you already know the output!


So in summary, Cython is basically just Python but faster (fast as fu-). It's a bit hard to configure but, definitely worth it for the 300x boost!
